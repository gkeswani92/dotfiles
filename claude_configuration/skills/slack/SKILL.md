# Slack

Search and interact with Slack via CLI.

## Invocation

```
/slack <command> [options]
```

## CLI Location

```bash
node ~/.claude/skills/slack/cli.js <command> [options]
```

## Commands

| Command | Description |
|---------|-------------|
| `user-info` | Get information about a Slack user |
| `evaluate-repl-function` | Execute JavaScript against the Slack API |
| `get-discussion-summary` | AI summary of discussions for a time period |
| `get-catchup-summary` | Summary of unread messages and DMs awaiting response |
| `get-technical-discussion-summary` | AI summary of technical discussions by project |
| `get-user-interactions` | Search interactions with another user |

## Examples

### Get User Info
```bash
# Current user
node ~/.claude/skills/slack/cli.js user-info

# Specific user
node ~/.claude/skills/slack/cli.js user-info --username john.doe
```

### Catch Up on Slack
```bash
# Get summary of unread messages
node ~/.claude/skills/slack/cli.js get-catchup-summary

# Without unanswered DMs
node ~/.claude/skills/slack/cli.js get-catchup-summary --include-unanswered false
```

### Discussion Summaries
```bash
# Today's discussions
node ~/.claude/skills/slack/cli.js get-discussion-summary

# Specific date range
node ~/.claude/skills/slack/cli.js get-discussion-summary --starting 2026-01-27 --ending 2026-01-31

# Technical discussions only
node ~/.claude/skills/slack/cli.js get-technical-discussion-summary --starting 2026-01-27
```

### User Interactions
```bash
# Find interactions with a user
node ~/.claude/skills/slack/cli.js get-user-interactions --username john.doe

# With date range
node ~/.claude/skills/slack/cli.js get-user-interactions --username john.doe --starting 2026-01-01

# With AI summary
node ~/.claude/skills/slack/cli.js get-user-interactions --username john.doe --summarize true
```

### Advanced: JavaScript REPL
```bash
# Search messages
node ~/.claude/skills/slack/cli.js evaluate-repl-function --function 'export default async (slack) => {
  const result = await slack.api.search.messages({
    query: "with:me after:2026-02-01",
    count: 10,
    sort_dir: "desc"
  });
  return result.messages.matches;
}'

# List channels
node ~/.claude/skills/slack/cli.js evaluate-repl-function --function 'export default async (slack) => {
  const result = await slack.api.conversations.list({ types: "public_channel,private_channel", limit: 50 });
  return result.channels.map(c => ({ id: c.id, name: c.name }));
}'

# Post a message
node ~/.claude/skills/slack/cli.js evaluate-repl-function --function 'export default async (slack) => {
  return await slack.api.chat.postMessage({
    channel: "C01234567",
    text: "Hello from CLI!"
  });
}'
```

## Search Query Syntax

When using `evaluate-repl-function` with search:

- `with:me` - Messages involving current user
- `with:@username` - Messages with specific user
- `after:2026-01-01` - Messages after date (exclusive)
- `before:2026-01-31` - Messages before date (exclusive)
- `on:2026-01-15` - Messages on specific day
- `during:january` - Messages during month
- `in:#channel` - Messages in specific channel

Combine with spaces for AND: `with:me with:@bob.dole after:2026-01-01`

## Output

Use `--output` to control format:
- `text` (default) - Human-readable
- `json` - JSON output
- `markdown` - Markdown formatted
- `raw` - Raw API response
